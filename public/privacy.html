Privacy Policy

Last updated: March 2, 2026

1. Overview

FormForge AI ("we", "our", "the app") is an open-source, AI-powered tool that helps users generate Google Forms, Google Drive drafts, and Google Calendar events from raw event descriptions. This Privacy Policy explains what data we access, how we use it, and how we protect it.

2. Data We Access

When you connect your Google account, we request access to the following scopes:

  • Google Forms API (forms.body) — to create and configure Google Forms on your behalf.
  • Google Drive API (drive.file) — to create folders and save reminder email drafts as text files. We can only access files created by this app.
  • Google Calendar API (calendar.events) — to create calendar events and reminder entries for event rounds.

3. Data We Do NOT Collect

We do NOT store, log, or transmit any of your personal data, Google account information, or event content to any external server or database. All processing happens in-session and is discarded when you close the app or disconnect.

  • We do not store your Google OAuth tokens on disk or in a database. Tokens are held in server-side session memory only.
  • We do not read, modify, or delete any pre-existing files in your Google Drive.
  • We do not access any Google Calendar data beyond events created by this app.
  • We do not collect analytics, tracking cookies, or usage telemetry.
  • We do not share your data with any third parties.

4. Third-Party Services

The app uses the following third-party services:

  • Groq API — Event text is sent to the Groq AI API (running Llama 3.3 70B) to generate form schemas and reminder email drafts. Groq's privacy policy applies to data processed by their API.
  • Google APIs — Google Forms, Drive, and Calendar APIs are used to create resources in your Google account. Google's privacy policy applies.

5. Session & Authentication

We use server-side sessions to temporarily store your Google OAuth tokens during your active session. These tokens are:

  • Stored in server memory only (not in a database).
  • Automatically cleared when the session expires or when you click "Disconnect".
  • Never logged, exported, or shared.

6. Data Retention

We retain zero persistent user data. When you disconnect your Google account or your session expires, all tokens and session data are permanently deleted from server memory. Files created in your Google Drive and Calendar remain in your account under your full control.

7. Security

We implement the following security measures:

  • HTTPS-only in production.
  • Helmet.js security headers.
  • CSRF protection via OAuth state parameter validation.
  • Rate limiting on all API endpoints.
  • Minimal OAuth scopes (drive.file instead of full Drive access).
  • Input validation and sanitization on all user inputs.

8. Your Rights

You can revoke FormForge AI's access to your Google account at any time:

  • Click the "Disconnect" button in the app header, or
  • Visit Google Account Permissions and remove FormForge AI.

9. Open Source

FormForge AI is open source. You can review the complete source code to verify our data handling practices.
